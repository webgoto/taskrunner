#taskrunner
Original node.js taskrunner for myself.

##usage 使い方

###install インストール
グローバル環境にインストール。

`$ npm install -g git+https://github.com/webgoto/taskrunner.git`

###setting 設定
* taskrunner.jsをプロジェクトフォルダのルートにコピー。
* taskrunner.jsを適宜、書き換える。

###start 起動
`$ node taskrunner.js`

###note 備考
* NODE_PATHにグローバルのnode_moduleフォルダを設定する
